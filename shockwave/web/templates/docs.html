Get started in two minutes.

Index a repo with Orbit, install Shockwave, and run your first blast-radius analysis.

1 · Index a repo with Orbit Local

Install the Orbit CLI, then index any repository into a local graph.

shell
# macOS / Linux
curl -fsSL https://gitlab.com/gitlab-org/orbit/\
knowledge-graph/-/raw/main/install.sh | bash

# Windows (PowerShell)
irm https://gitlab.com/gitlab-org/orbit/\
knowledge-graph/-/raw/main/install.ps1 | iex

orbit index .

2 · Install Shockwave

Python 3.11+. The web UI is an optional extra.

shell
pip install -e .

# with the Blast Monitor web UI
pip install -e ".[web]"

3 · Analyze

shell
# impact of a symbol or file
shockwave analyze setupmethod --format md

# blast radius of a whole diff (great in CI)
shockwave diff main

# against the hosted graph — Orbit Remote
shockwave analyze compute --remote https://gitlab.com --token $GITLAB_TOKEN

# the interactive map
shockwave-web        # → http://127.0.0.1:7777
🤖

Add the MR bot

Drop the provided .gitlab-ci.yml in your project and add a masked GITLAB_TOKEN variable (api scope). On every merge request, the bot posts the blast-radius review. Orbit must be enabled on the group.
